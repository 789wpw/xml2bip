
Usage:
	1: run gene_models.bash to translate bip to smv file, e.g., ./gene_models.bash
	2: run add_property.py to add property to the smv file, e.g., python add_property.py
	3: run nuXmv_ic3.bash to verify the smv file, e.g., ./nuXmv_ic3.bash
	4: run clean_up.bash to remove intermediate files (optional)

Note:
	DO NOT remove file: nuXmv_ic3.cmd AND BIP2NUSMV.jar
